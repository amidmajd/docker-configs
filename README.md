## Docker Compose For PostgresDB

1. Run `docker-compose up` to start the container
2. Go to `127.0.0.1:5050`
3. Login to PgAdmin :
   - **Email** : `mail@gmail.com`
   - **password** : `password`
4. Add server in Admin panel:
   - Name : `anything`
   - Host name/address : `postgres`
   - Username : `admin`
   - Password : `password`
- Accessing DataBase:
    - Address : `localhost`
    - Port : `5432`
    - Username : `admin`
    - Password : `password`
